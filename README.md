# Fake-URL-Detector
Website URL Safety Checker is a Python-based GUI application developed for the Introduction to Programming module at Softwarica College of IT &amp; E-Commerce in collaboration with Coventry University. It classifies URLs as Safe, Suspicious, or Phishing using rule-based detection techniques.
# Fake URL Detector

This project is a simple Fake URL Detector (Phishing Detection Tool)
developed using Python and Tkinter.

## Features
- Detects Safe, Suspicious, and Phishing URLs
- Rule-based logic only
- Detects fake domains, IP-based URLs, long URLs, and missing HTTPS
- Simple GUI using Tkinter

## Technologies Used
- Python
- Tkinter

## How to Run
1. Open the project folder
2. Run main.py using Python
3. Enter a URL and click "Check URL"
